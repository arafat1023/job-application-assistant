# Job Application Assistant

A toolkit for automated job searching, application tracking, and interview preparation.

## Features

- **Job Search Tools**: Automated tools for searching multiple job boards
- **Application Management**: Track and manage job applications
- **Document Generation**: Generate tailored CVs and cover letters

## Installation

```bash
git clone https://github.com/arafat1023/job-application-assistant.git
cd job-application-assistant
npm install
```

## Usage

See [SETUP.md](SETUP.md) for detailed setup instructions.

## Job Board Tools

- **HN Hiring Search**: Search Hacker News monthly hiring threads
- **Freehire Search**: Search Freehire board for remote opportunities
